Css Tutorial: Shorthand Properties

 

Toufiq Nuur on 07 Mar 2021

#css

Shorthand atau dalam bahasa Indonesia disebut Stenografi adalah cara menulis ringkas dan cepat yang biasa dipakai untuk menyalin pembicaraan.

Lalu apa itu css shorthand?

Css shorthand merupakan cara penulisan kode css dengan lebih singkat dan rapi.

Shorthand Properties

Css memiliki yang namanya shorthand properties yang membuat kita dapat menuliskan css rule dalam satu baris.

Contohnya background merupakan shorthand properties yang dapat menampung value dari background-color, background-image, background-repeat, dan background-position.

Studi Kasus

Studi kasus yang pertama misalkan kita ingin mengatur margin sebuah halaman. Berikut kode yang digunakan.

.element {
  margin-top: 1rem;
  margin-left: 1rem;
  margin-bottom: 1rem;
  margin-right: 1rem;
}

Dengan shorthand properties kita bisa membuatnya lebih rapi dan ringkas seperti dibawah ini.

.element {
  margin: 1rem;
}

Properti margin dapat diisi dengan beberapa variasi sebagai berikut.

  • Jika hanya diisi dengan 1 value, maka itu mendefinisikan keempat sisinya. margin: 20px artinya semua sisi marginnya 20px.
  • Diisi dengan 2 value, artinya value pertama mendefinisikan margin top dan bottom sedangkan value kedua mendefinisikan margin right dan left.
  • Diisi dengan 3 value, maknanya value pertama mendefinisikan margin top, yang kedua margin right dan left yang terakhir margin bottom.
  • Dan jika diisi dengan empat value, maka value pertama margin-top, yang kedua margin-left, yang ketiga margin bottom, dan terakhir margin-right

Kesimpulan

Shorthand properties membuat kita lebih mudah untuk menuliskan css rule. Selain itu, berkat shorthand properties css kita terlihat lebih ringkas dan rapi. Namun ada beberapa kasus yang membuat kita tidak bisa menghasilkan shorthand properties.
