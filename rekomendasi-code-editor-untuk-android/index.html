Rekomendasi Code Editor Untuk Android

Toufiq Nuur on 13 Jan 2021

#setup #android #development

Masalah utama bagi pengguna smartphone yang ingin belajar web development adalah keterbatasan environment. Environment disini mencakup IDE, Code Editor, dll.

Selain itu seorang web developer harus mampu membuat tampilan web yang responsive diberbagai device. Nah bagi yang hanya bermodalkan smartphone tentunya akan sulit untuk mengatasi yang satu ini, ukuran layar yang kecil.

Namun, jangan khawatir semakin hari teknologi berkembang, android pun begitu. Sekarang banyak aplikasi-aplikasi yang dapat membantu kita membangun website baik itu statis maupun dinamis.

Code Editor untuk android

Sebenarnya banyak code editor yang tersedia untuk android, namun kita harus memilah mana yang nyaman untuk digunakan baik dari segi UI maupun experience. Disini telah saya rangkum code editor yang recommended.

Acode

Yang pertama adalah acode, code editor yang satu ini adalah yang paling recommended (menurut saya). Dari beberapa code editor yang ada menurut saya ini yang paling nyaman UInya cukup bagus, mudah digunakan dan ringan.

Kelebihan

Dilengkapi dengan ftp dan github sehingga dapat mengedit file dan langsung menyimpannya secara online. Dilengkapi dengan tema untuk editornya serta tema untuk aplikasi. Terdapat shortcut yang dapat diakses dengan kombinasi tombol.

Kekurangan

Perlu menggunakan versi pro untuk mengubah tema aplikasi.

Unduh: Acode

anWriter

Kode editor ini cocok digunakan untuk smartphone keluaran lama yang mempunyai memori kecil. Ukuran aplikasi ini kecil dan ringan dijalankan.

Kelebihan

Mempunyai ukuran yang kecil, ringan dijalankan terdapat tema untuk editor, autocomplete, built in webview. Dapat dikoneksikan dengan FTP Server.

Kekurangan

UInya terlalu standar, warnanya componentnya kurang menarik. Tema editornya hanya tersedia untuk versi pronya saja.

Unduh: anWriter

Spck Editor

Code editor ini rekomendasi kedua setelah acode (pendapat pribadi). Dari segi UI menurut saya lebih bagus dari Acode namun dari segi experience pemakai cenderung lebih nyaman Acode.

Kelebihan

UI jelas lebih bagus dan menarik, terintegrasi dengan github. Terdapat tema yang beragam, dilengkapi dengan emmet (hanya html).

Kekurangan

Pengelolaan file kurang leluasa.

Unduh: Spck Editor

VHEditor

Dan yang terakhir ini mempunyai tampilan yang mirip dengan VSCode karena menggunakan code server. Nyaman digunakan di tablet dan menggunakan external keyboard.

Kelebihan

Nyaman digunakan fiturnya lengkap mirip VSCode. Terdapat banyak plugin, terintegrasi dengan git. Banyak kustomisasi tema dan icon. Built in terminal.

Kekurangan

Tidak bisa menginstal program seperti nodejs dengan perintah pkg atau apt. Terminal merupakan termux yang dimodifikasi. Cukup memakan ram dan baterai.

Unduh: VHEditor

Kesimpulan

Dari semua code editor diatas yang paling saya rekomendasikan adalah Acode. Karena menurut saya itulah yang paling mudah dan nyaman digunakan. Code editor apa yang cocok untuk anda? Balik lagi ke diri masing karena setiap orang mempunyai selera yang berbeda cukup menyesuaikan.
