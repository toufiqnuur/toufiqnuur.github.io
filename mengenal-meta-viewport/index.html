Mengenal Meta Viewport

Toufiq Nuur on 15 Jan 2021

#css #html #responsive

Viewport adalah area dari window dimana konten akan terlihat. Viewport digunakan untuk membuat web yang responsif. Yang artinya akan menyesuaikan dengan perangkat yang digunakan oleh user.

<meta name="viewport" content="width=device-width, initial-scale=1">

Kode diatas merupakan penerapan meta viewport yang lazim digunakan oleh web developer. Nilai width=device-width maknanya lebar halaman akan mengikuti lebar layar perangkat yang digunakan. Nilai initial-scale=1 artinya halaman akan diberi skala (zoom) 1.

Pengaruh meta ini akan terlihat jika kita membuka halaman web kita di perangkat yang mempunyai layar kecil (smartphone). Baiklah langsung saja kita uji dengan kode dibawah ini.

<!DOCTYPE html>
<html>
  <head>
    <title>Tanpa Meta Viewport</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

Hasilnya akan terlihat seperti ini.

Bagaimana jika menggunakan meta viewport? Mari kita uji dengan kode dibawah ini.

<!DOCTYPE html>
<html>
  <head>
    <title>Dengan Meta Viewport</title>
    <meta name="viewport" content="width=device-width, initial-scale=1">
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

Hasilnya akan terlihat seperti ini.

Kesimpulan

Jadi kesimpulannya meta viewport sangat penting untuk membuat halaman website menjadi responsive di berbagai perangkat. Sehingga akan meningkatkan experience bagi user.
