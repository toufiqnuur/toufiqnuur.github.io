Menginstal Dan Menjalankan Phpmyadmin Di Android Dengan Termux

Toufiq Nuur on 28 Feb 2021

#phpmyadmin #php #mariadb #termux

PHPmyadmin kalian pasti sudah tahu dmbs yang satu ini. Dmbs yang satu ini cukup populer dikalangan programmer.

Bagi yang belum tahu, phpmyadmin merupakan dbms atau database management system berbasis web untuk mengelola database MySql.

Lantas apa sih fitur-fiturnya, sehingga membuatnya begitu populer?

  • Antarmuka berbasis web.
  • Impor data dari csv ke sql
  • Ekspor data ke berbagai format: CSV, SQL, XML, PDF, ISO / IEC 26300 - OpenDocument Text dan Spreadsheet, Word, Excel, LATEKS dll.
  • Membuat grafik PDF dari tampilan basis data anda.
  • Membuat kompleks query menggunakan Query-by-example. (QBE)
  • Pencarian global dalam basis data
  • Dan masih banyak lagi.

Bagaimana fitur-fitur yang ditawarkan? Menarik bukan? Jika kalian tertarik dan ingin mencobanya mari kita coba instalasi phpmyadminnya.

Tunggu dulu, sebelum kita melakukan instalasi kita penuhi dulu requirement yang dibutuhkan yaitu sebagai berikut.

  • PHP 7.2.5 atau lebih baru
  • Mariadb 5.5 atau lebih baru

Bagaimana sudah menyiapkan requirement yang diperlukan? Jika sudah yuk lanjut ke tahap instalasi phpmyadmin.

1. Download phpmyadmin

Untuk mendownload phpmyadmin caranya cukup mudah, kalian bisa mendapatkannya disitus resminya (https://www.phpmyadmin.net/).

Jika sudah ekstrak file yang sudah didownload ke folder yang kalian inginkan.

2. Jalankan MySql Server

Selanjutnya buka termux atau aplikasi terminal favorit kalian. Kemudian untuk menjalankan MySql servernya masukan perintah mysqld_safe.

3. Jalankan phpmyadmin

Disini kita akan memakai web server bawaan PHP untuk menjalankan phpmyadmin. Caranya pertama arahkan ke folder phpmyadminnya, setelah itu masukan perintah php -S localhost:8080. Buka browser favorit kalian kemudian masukan url localhost:8080.

Jika sudah berhasil masuk ke tampilan login, selanjutnya tinggal masukan username dan password disini bawaan username dan password dari mariadbnya root.

Jika berhasil maka tampilannya harusnya seperti ini.
