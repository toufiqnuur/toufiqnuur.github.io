Looping Array Dengan for in Dan for Of

Untuk mendapatkan nilai atau value dari sebuah array atau objek kita biasanya menggunakan perulangan for. Seperti dibawah ini: const buah = ["mangga","jeruk","apel","nanas"] for (let i=0; i<buah.length; i++) { console.log(buah[i]) } //result // 'mangga' // 'jeruk' // 'apel' // 'nanas' Kode diatas memang terlihat normal dan tidak ada yang salah. Namun menurut saya agak ribet karena kita harus membuat variabel baru, menentukan jumlah looping dll. Saya lebih cenderung menyukai menggunakan for .

Mengenal Meta Viewport

Viewport adalah area dari window dimana konten akan terlihat. Viewport digunakan untuk membuat web yang responsif. Yang artinya akan menyesuaikan dengan perangkat yang digunakan oleh user. <meta name="viewport" content="width=device-width, initial-scale=1"> Kode diatas merupakan penerapan meta viewport yang lazim digunakan oleh web developer. Nilai width=device-width maknanya lebar halaman akan mengikuti lebar layar perangkat yang digunakan. Nilai initial-scale=1 artinya halaman akan diberi skala (zoom) 1. Pengaruh meta ini akan terlihat jika kita membuka halaman web kita di perangkat yang mempunyai layar kecil (smartphone).
