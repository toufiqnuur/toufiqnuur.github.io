Php Tutorial: Instalasi Php Composer di termux

 

Toufiq Nuur on 25 Feb 2021

#PHP

Oke sebelum masuk ke tutorialnya kita bahas terlebih dahulu apa sih composer itu.

Composer adalah sebuah tool untuk mengelola dependency di PHP. Kalau di javascript composer itu kita kenal sebagai NPM.

System Requirements

  • PHP 5.3.2+

Instalasi

  • Install PHP

    Bagi yang sudah instal PHP skip ke step selanjutnya, jika belum simak caranya. Ketikan perintah berikut.

    pkg install php

  • Instal Composer

    Untuk menginstal composer kita harus melalui 4 tahapan diantaranya.

    • Mendownload installer

      Untuk mendownload installernya caranya cukup masukan perintah berikut.

      php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"

    • Memverifikasi installer

      Kemudian untuk memverifikasi installernya masukan perintah berikut.

      php -r "if (hash_file('sha384', 'composer-setup.php') === '756890a4488ce9024fc62c56153228907f1545c228516cbf63f885e036d37e9a59d27d63f46af1d4d07ee0f76181c7d3') { echo 'Installer verified'; } else { echo 'Installer corrupt'; unlink('composer-setup.php'); } echo PHP_EOL;"

    • Menjalankan installer

      Setelah itu kita jalankan installernya dengan perintah.

      php composer-setup.php --install-dir=$PREFIX/bin --filename=composer

    • Menghapus installer Terakhir kita hapus installernya dengan perintah.

      php -r "unlink('composer-setup.php');"

  • Cek installasi Langkah terakhir kita pastikan instalasi berhasil atau tidak dengan cara berikut ini.

    u0_a273@localhost:~ composer --version
Composer version 2.0.9 2021-01-27 16:09:27
u0_a273@localhost:~

Nah begitulah cara instal PHP composer di android dengan bantuan termux.

Referensi: https://getcomposer.org/download/
