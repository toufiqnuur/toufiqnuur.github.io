JavaScript Tutorial: Looping for ...in dan for ...of

 

Toufiq Nuur on 23 Jan 2021

#javascript

Untuk mendapatkan nilai atau value dari sebuah array atau objek kita biasanya menggunakan perulangan for. Seperti dibawah ini:

const buah = ["mangga","jeruk","apel","nanas"]
for (let i=0; i<buah.length; i++) {
console.log(buah[i])
}
//result
// 'mangga'
// 'jeruk'
// 'apel'
// 'nanas'

Kode diatas memang terlihat normal dan tidak ada yang salah. Namun menurut saya agak ribet karena kita harus membuat variabel baru, menentukan jumlah looping dll.

Saya lebih cenderung menyukai menggunakan for ...in atau for ...of. Agar lebih jelas bandingkan kode diatas dengan kode dibawah ini.

const buah = ["mangga","jeruk","apel","nanas"]
for (let nama of buah) {
console.log(nama)
}
//result
// 'mangga'
// 'jeruk'
// 'apel'
// 'nanas'

Lebih rapi dan simpel bukan? Tapi menggunakan perulangan for ...of seperti diatas memiliki kekurangan yaitu hanya dapat mengambil value atau nilai dari array, jadi kita tidak bisa mengambil indexnya.

Selanjutnya ada for ...in dengan ini kita bisa mendapatkan index beserta value dari array. Perhatikan kode dibawah.

const buah = ["mangga","jeruk","apel","nanas"]
for (let index in buah) {
console.log(buah[index])
}
//result
// 'mangga'
// 'jeruk'
// 'apel'
// 'nanas'

Kesimpulan

Jadi kesimpulannya for ...of hanya mengembalikan value, sementara for ...in mengembalikan index.

Jadi masih betah dengan looping array dengan for (let i; i<array.length; i++ ?
